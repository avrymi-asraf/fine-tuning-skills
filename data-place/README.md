# Data Place

Research articles and papers related to fine-tuning language models.

## Index

See `index.json` for searchable catalog of all articles.

## Directory Structure

```
data-place/
├── raw/           # Original PDF files
├── index.json     # Searchable article index
└── README.md      # This file
```

## Adding New Articles

1. Place PDF in `raw/` directory
2. Update `index.json` with metadata
3. Include: title, source, topics, date added
